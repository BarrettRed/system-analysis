# Task 1. Матрицы смежности для предикатов

## Описание задания

В данном задании реализуется функция для построения пяти матриц смежности, каждая из которых соответствует определённому предикату (отношению) на дереве.

### Отношения

1. **r1** — непосредственное управление  
   (x управляет y, если x является родителем y)

2. **r2** — непосредственное подчинение  
   (x подчиняется y, если x является ребёнком y)

3. **r3** — опосредованное управление  
   (x управляет y, если x является предком y, но не родителем)

4. **r4** — опосредованное подчинение  
   (x подчиняется y, если x является потомком y, но не ребёнком)

5. **r5** — соподчинение  
   (x и y соподчинены, если они имеют одного родителя и находятся на одном уровне)

## Формат входных данных

Функция принимает:

- `s: str` — CSV-строка, содержащая список рёбер ориентированного дерева.  
  Формат: `"1,2\n1,3\n3,4\n3,5\n5,6\n6,7"`

- `e: str` — идентификатор корневого узла.

## Формат выходных данных

Функция возвращает кортеж из пяти матриц смежности в виде списка списков `bool`:

```python
(
    r1,  # матрица для непосредственного управления
    r2,  # матрица для непосредственного подчинения
    r3,  # матрица для опосредованного управления
    r4,  # матрица для опосредованного подчинения
    r5   # матрица для соподчинения
)
```

Каждая матрица имеет размер n x n, где n — количество вершин в дереве.
Вершины индексируются в порядке возрастания их номеров.
